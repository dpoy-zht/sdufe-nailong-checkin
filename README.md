# 🐉 奶龙图书馆远程签到系统

> **让奶龙替你签到，你只管摸鱼** 🐉✨
> **一个大学生为了少走两步路，花了三天做出来的签到神器**

## 🚨 这是个啥？

山东财经大学（SDUFE）图书馆签到系统。**纯前端，零依赖，打开即用**。

核心原理：获取图书馆服务器时间戳 → 拼接签到URL → 手动跳转完成签到。

就这么简单。简单到奶龙都会。简单到我的室友看完代码说"就这？"然后默默掏出了手机开始用。

> 为什么不自动跳转？因为微信浏览器的 Cookie 会丢。
> 为什么 Cookie 会丢？因为微信。
> 为什么用微信？因为图书馆签到系统嵌在微信里。
> 这就是蝴蝶效应。🦋

## 🎯 在线体验

**https://dpoy-zht.github.io/sdufe-library-checkin/**

> ⚠️ 必须连接 **SDUFE 校园网**！否则奶龙的法力范围覆盖不到校外。
>
> 💡 **首次访问如果看到旧版本**，请用**无痕模式**打开，或者访问 `clear.html` 一键清缓存。
> 这是 Service Worker 的锅，不是奶龙的锅。

## 🎨 功能大赏（v2.5 全功能版）

你以为这只是个签到页面？不，这是一个**奶龙主题公园**。

### 🎭 1. 奶龙情绪系统
奶龙会根据签到状态**实时变脸**，比你前任的情绪稳定多了。

| 状态 | 奶龙表情 | 奶龙OS |

| 临时离馆 | （保持不变） | "快回来，奶龙一个人害怕" |

### 🎙️ 2. 奶龙语音播报
签到后奶龙会开口说话。用 `speechSynthesis` 实现，**零音频文件**。

| 操作 | 奶龙语音 | 情绪分析 |
|------|---------|---------|
| 签到入馆 | "签到成功啦~奶龙陪你学习！" | 激动，像中了500万 |
| 临时离馆 | "临时离馆~记得早点回来哦！" | 焦虑，怕你跑了 |

> ⚠️ 音色取决于你的设备系统。奶龙在安卓上可能是萝莉音，在 iOS 上可能变成大叔音。这不叫 bug，这叫**跨平台多样性**。
> ⚠️ 如果没听到声音，请确认没有静音。奶龙再怎么激动也发不出静音。

### 📱 9. PWA 支持
添加到手机桌面后，它就像一个**原生 App**：

- 📲 桌面图标：奶龙头像
- 🌙 独立窗口：没有浏览器地址栏，沉浸体验
- ⚡ 离线缓存：断网也能打开页面（虽然签到需要网络）
- 📦 缓存策略：network-first，永远加载最新版本（踩过 cache-first 的坑了）

> 为什么加 PWA？因为奶龙值得被添加到桌面。
> 不添加？奶龙会伤心。奶龙虽然不会说，但你知道的。

## 📡 API 原理（给技术人的）

```
1. GET https://libst.sdufe.edu.cn/api.php/v3qrtime
   返回：{"status":1,"msg":"服务器时间","data":51526269564}
   （这个时间戳不是标准 Unix 时间戳，别问为什么，图书馆说的算）

2. 拼接签到URL：
   http://www.skalibrary.net/wx/scancheck?school=sdufe&type={1|2|3}&t={时间戳}
   type=1 → 签到入馆
   type=2 → 临时离馆
   type=3 → 签离出馆

3. 手动点击跳转到微信浏览器完成签到
   （自动跳转会丢 Cookie，这是血泪教训）
```

> 没错，整个签到流程的核心就是拼 URL。就是这么朴实无华且枯燥。
> 但就是这么简单的东西，奶龙给它包了 14 层功能。

## 🛠️ 技术架构

```
index.html     ← 整个项目的全部（没错就一个文件）
├── 马卡龙奶油卡通风 CSS 变量系统
├── 磨砂玻璃卡片（backdrop-filter:blur，Safari 可能会骂我）
├── CSS-only 动画（云朵、星星、粒子、火焰、浮动）
├── 时段主题切换（6个时段 + 暗黑模式）
├── 奶龙情绪系统（localStorage 状态持久化）
├── Web Speech API 语音播报（零音频文件）
├── Confetti 签到撒花（40片随机纸屑）
├── 连续签到 streak 计数器（日期差计算）
├── 签到时间线可视化（竖向时间轴）
├── 成就系统（9个成就 + 3×3网格）
├── 空闲催促气泡（30秒超时检测）
├── 长按彩蛋检测（1500ms 长按事件）
├── 奶龙语录轮播（15条随机语录）
├── 季节粒子特效（emoji粒子系统）
├── 22:00 闭馆提醒（定时检测）
└── 滚动跟随奶龙（固定定位 + transform）

sw.js         ← Service Worker（network-first 缓存策略）
manifest.json ← PWA 清单（奶龙图标 + 主题色）
```

**技术栈：** 纯原生 HTML/CSS/JS，**零依赖**。连 jQuery 都没有。vanilla 到不能再 vanilla。
**构建工具：** 记事本（开玩笑的，是 VS Code）。
**部署方式：** GitHub Pages（免费的白嫖，奶龙很满意）。

## 📱 使用方法

1. 📶 手机连接 **SDUFE WiFi**
2. 🌐 打开 https://dpoy-zht.github.io/sdufe-library-checkin/
3. ✅ 看到「已连接」= 奶龙法力已就位
4. 🎯 点你想点的按钮（签到/暂离/签离）
5. 🔊 听奶龙说一句鼓励的话（记得开声音！）
6. 🥳 看奶龙变脸 + 五彩纸屑特效
7. 📋 弹出 URL 后点击「跳转到图书馆」
8. 🤖 在微信浏览器里完成签到
9. 🏆 查看成就面板，看看今天解锁了什么
10. 🐉 长按奶龙发现彩蛋

## ⚠️ 已知问题（别怪奶龙）

- [ ] 离开校园网就废了（奶龙又不是5G基站）
- [ ] `libst.sdufe.edu.cn` 偶尔抽风返回「异常访问」（刷新就完事了）
- [ ] 微信浏览器外签到可能失败（Cookie 问题，老传统了）
- [ ] Service Worker 缓存可能导致看到旧版本（已改为 network-first，应该好了）
- [ ] 奶龙图片位置调整了至少 **12 次**（这不是 bug，这叫「奶龙精确定位系统 v12.0」）
- [ ] GitHub PAT token 在对话中暴露了 **N 次**（已经不想数了，建议 revoke）
- [ ] TTS 语音在部分浏览器被静默拦截（Apple 说这叫"隐私保护"）
- [ ] 奶龙语音可能变成大叔音（取决于你的设备系统）
- [ ] 成就系统有人类能解锁「传奇学神」吗？（存疑）
- [ ] 长按彩蛋的文案越来越离谱了（但奶龙说这就是它的风格）
- [ ] 季节粒子在冬天可能引起赛博感冒（建议穿件虚拟外套）
